# elevator-block

## Generate assets

```bash
./gradlew runData
```

## Build into JAR

**Generate assets before build**

```bash
./gradlew build
```

The built JAR can be found in `build/reobjJar` as `output.jar`.

## Run test client/server

Client : 

```bash
./gradlew runClient
```

Server :

```bash
./gradlew runServer
```
